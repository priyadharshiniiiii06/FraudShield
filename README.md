# FraudShield

## Project Title
FraudShield - Fraud Detection and Prevention System

## Description
FraudShield is a web application developed using Python and Flask to detect and analyze fraudulent transactions.

## Features
- User-friendly interface
- Fraud detection analysis
- Transaction monitoring
- Risk assessment
- Secure data handling

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Machine Learning
